# @workspace/resupply-telecom

Vendor adapters for SMS (Twilio), Voice (Twilio), and Email (SendGrid). Phase 0 ships only the interfaces and mock implementations — real Twilio/SendGrid wiring lands in Phase 3. May only be imported by the resupply api and worker apps.

## Status

Phase 0 — scaffolding only. No exports yet. See `docs/resupply/adr/` for the
architectural decisions that govern this package.
